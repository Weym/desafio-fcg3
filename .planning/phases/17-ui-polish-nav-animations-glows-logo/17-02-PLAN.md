<objective>
Retune glow effect colors and glassmorphism fill/border for light mode so the Cyber-Academic aesthetic reads correctly on light backgrounds.

Purpose: The current neon glow colors (#00E5FF Electric Teal) have insufficient contrast against light backgrounds, making glows look like rendering artifacts. Glassmorphism uses 5% white fill + 12% white border which is invisible on white backgrounds. This plan adds light-mode neon variants and adapts the glassmorphism treatment.

Output: Light mode that feels intentionally "cyber-academic" with readable glow contrast and visible glassmorphism borders.
</objective>
