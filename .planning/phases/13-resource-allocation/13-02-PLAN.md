<!-- Backend API contracts from Plan 01 that this plan consumes -->

From backend/src/features/resources/schemas.py (created in Plan 01):
```
GET /resources → List[ResourceResponse]
  Query params: resource_type (optional), is_available (optional)
  
POST /resources → ResourceResponse (201)
  Body: { name, resource_type, capacity?, location?, description?, requires_authorization? }
  
PUT /resources/{id} → ResourceResponse
  Body: { name?, resource_type?, capacity?, location?, description?, requires_authorization? }
  
DELETE /resources/{id} → ResourceResponse (soft-delete → is_available=false)
```

ResourceResponse shape:
```json
{
  "id": "uuid",
  "name": "string",
  "resource_type": "room|lab|equipment|auditorium|study_room|sports_court",
  "capacity": int | null,
  "location": "string" | null,
  "description": "string" | null,
  "requires_authorization": bool,
  "is_available": bool,
  "created_at": "datetime"
}
```

From mobile established patterns:
```dart
// Service injection pattern:
class StaffResourceService {
  final DioClient _client;
  StaffResourceService({required DioClient client}) : _client = client;
}

// Provider pattern:
@Riverpod(keepAlive: true)
StaffResourceService staffResourceService(Ref ref) {
  final client = ref.watch(dioClientProvider);
  return StaffResourceService(client: client);
}
```
