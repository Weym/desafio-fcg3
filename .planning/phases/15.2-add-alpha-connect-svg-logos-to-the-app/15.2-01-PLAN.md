**Key design decisions:**
- When `showText` is true AND `size > 40`: render `alpha_connect_logo.svg` (full logo with α mark + ALPHA + CONNECT text)
- When `showText` is false OR `size <= 40`: render `alpha_connect_shortlogo.svg` (α mark only)
- The `showTagline` parameter: the full SVG already contains the tagline text in the `#phrase` group. Since the SVG tagline is baked in, when `showTagline` is false we use `colorFilter` on the full SVG but the tagline is always present in the full logo SVG. This is acceptable — the tagline is small and matches the brand design. If the caller wants no tagline and no text, they pass `showText: false` which uses the short logo.
- Theme-adaptive coloring: use `ColorFilter.mode(effectiveColor, BlendMode.srcIn)` to tint the entire SVG to the theme's primary color (or the override `color` parameter). This replaces all fill colors in the SVG with the single theme color.

**Complete replacement code for `alpha_connect_logo.dart`:**

```dart
import 'package:flutter/material.dart';
import 'package:flutter_svg/flutter_svg.dart';

/// Theme-adaptive Alpha Connect branding widget.
/// Renders SVG logos from assets with theme-aware color tinting.
/// Full logo (α + ALPHA CONNECT text) or short logo (α mark only).
class AlphaConnectLogo extends StatelessWidget {
  const AlphaConnectLogo({
    super.key,
    required this.size,
    this.color,
    this.showText = true,
    this.showTagline = false,
  });

  /// Height of the logo widget.
  final double size;

  /// Override color for the logo. Defaults to `Theme.of(context).colorScheme.primary`.
  final Color? color;

  /// Whether to show the full logo (α + text). Auto-disabled when [size] <= 40.
  final bool showText;

  /// Whether to show the tagline (only relevant with full logo).
  final bool showTagline;

  @override
  Widget build(BuildContext context) {
    final effectiveColor = color ?? Theme.of(context).colorScheme.primary;
    final useFullLogo = showText && size > 40;

    final assetPath = useFullLogo
        ? 'assets/logos/alpha_connect_logo.svg'
        : 'assets/logos/alpha_connect_shortlogo.svg';

    return SvgPicture.asset(
      assetPath,
      height: size,
      colorFilter: ColorFilter.mode(effectiveColor, BlendMode.srcIn),
      semanticsLabel: 'Alpha Connect Logo',
    );
  }
}
```

**IMPORTANT:** Remove the `google_fonts` import — it is no longer needed in this file. The `const` constructor is preserved. The `showTagline` parameter is kept for API compatibility but the full SVG always includes the tagline text (this is the designed brand asset).
