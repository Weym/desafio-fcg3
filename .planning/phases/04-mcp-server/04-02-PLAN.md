All tools must import `mcp` from `mcp_server.main` — use the pattern of getting the mcp instance and registering tools on it. The simplest approach: each tool module receives the `mcp` instance via a `register(mcp)` function, or tools import `mcp` from main.

**Recommended pattern** (to avoid circular imports):
- `main.py` creates `mcp = FastMCP(...)`.
- Tool modules define functions that accept `mcp` and register tools:
  ```python
  def register_student_tools(mcp: FastMCP):
      @mcp.tool(...)
      async def get_student_info(...):
          ...
  ```
- `main.py` calls `register_student_tools(mcp)` etc.

This avoids circular imports and keeps tool modules clean.
