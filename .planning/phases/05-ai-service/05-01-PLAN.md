2. `load_chat_history(pool, session_id: str, k: int = 20) -> list`:
   Query `chat_messages` table: `SELECT role, content FROM chat_messages WHERE chat_session_id = %s ORDER BY created_at DESC LIMIT %s`.
   Convert to LangChain message types:
   - `role == 'user'` → `HumanMessage(content=content)`
   - `role == 'assistant'` → `AIMessage(content=content)`
   - `role == 'system'` → `SystemMessage(content=content)`
   Return messages in chronological order (reverse the DESC result).
   Import from `langchain_core.messages`.

3. `save_chat_message(pool, session_id: str, role: str, content: str) -> None`:
   Insert into `chat_messages`: `INSERT INTO chat_messages (chat_session_id, role, content) VALUES (%s, %s, %s)`.
   Per D-09, the AI service writes the assistant's response after generation.

4. `check_db_health(pool) -> bool`:
   Run `SELECT 1` and return True/False.

**ai_service/llm_factory.py** — per D-07 and agent's discretion for LLM factory:
Create `create_llm(settings) -> BaseChatModel`:

```python
def get_model_string(settings) -> str:
    """Build the provider:model string for create_agent."""
    provider = settings.LLM_PROVIDER
    model = settings.LLM_MODEL
    if provider == "openai":
        return f"openai:{model}"
    elif provider == "gemini":
        return f"google_genai:{model}"
    else:
        raise ValueError(f"Unsupported LLM provider: {provider}. Use 'openai' or 'gemini'.")
```

Also export `get_model_string` — the agent creation (Plan 04) will use it with `create_agent(model=get_model_string(settings), ...)`.

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| FastAPI backend → AI Service | Internal HTTP call; AI service trusts caller for session validity (D-04) |
| AI Service → PostgreSQL | Direct DB access for chat history; SQL injection risk |
| AI Service → MCP Server | Tool calls via streamable-http; session_id in header |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01 | Injection | database.py SQL queries | mitigate | All queries use parameterized %s placeholders with psycopg3 — never string interpolation |
| T-05-02 | Information Disclosure | OPENAI_API_KEY | mitigate | Environment variable only; never in source code or logs |
| T-05-03 | Information Disclosure | LLM responses | accept | LLM may hallucinate; system prompt rule #7 mitigates but cannot eliminate |
| T-05-04 | Denial of Service | Agent execution time | mitigate | max_execution_time=45s and max_iterations=10 (D-07) prevent runaway agent loops |
</threat_model>
