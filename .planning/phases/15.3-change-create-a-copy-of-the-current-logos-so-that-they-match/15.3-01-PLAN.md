<objective>
Create light-mode and dark-mode SVG logo variants and update the AlphaConnectLogo widget to select the correct variant based on the current theme brightness.

Purpose: The current logo uses `ColorFilter.mode(primary, BlendMode.srcIn)` which tints the entire SVG to a single color (Electric Teal #00E5FF). This makes the logo monochrome and loses the multi-element structure of the original design (α mark vs text vs tagline). This phase creates proper brightness-adaptive SVG variants with natural fill colors that provide high contrast on their respective backgrounds, while removing the single-color tint.

Output: 4 new SVG files (light/dark × full/short), updated widget, removed old SVGs
</objective>
