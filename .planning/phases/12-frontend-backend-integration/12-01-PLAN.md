2. Update `docker-compose.yml` `fastapi-app` service:
   - Add an entrypoint script pattern: after uvicorn starts, the seed runs once
   - Actually, better approach: add a new one-shot service `seed` that runs after postgres is healthy:
     ```yaml
     seed:
       build: ./backend
       container_name: fcg3-seed
       command: python -m scripts.seed
       environment: (same DB vars as fastapi-app)
       depends_on:
         postgres:
           condition: service_healthy
         fastapi-app:
           condition: service_healthy
       networks:
         - data-network
       restart: "no"
       profiles:
         - seed
     ```

   - First-time setup: `docker compose --profile seed up` or `docker compose run seed`
