**Strategy:** GoRouter's `pageBuilder` property on `GoRoute` allows specifying `CustomTransitionPage`. Two transition types:

1. **Fade-through (300ms)** for same-level tab navigation — routes that are direct children of ShellRoute (client home, chat, documents, notifications, resources, support, and all staff top-level tabs)
2. **Horizontal slide (250ms)** for push/detail routes — routes nested under tab routes (chat detail `:sessionId`, appointment detail, intervention chat)

**Implementation:**

Create two helper functions at the bottom of the file (before the closing `}`):

```dart
CustomTransitionPage<void> _fadeThroughPage({
  required LocalKey key,
  required Widget child,
}) {
  return CustomTransitionPage<void>(
    key: key,
    child: child,
    transitionDuration: AppAnimations.fadeThroughDuration,
    reverseTransitionDuration: AppAnimations.fadeThroughDuration,
    transitionsBuilder: (context, animation, secondaryAnimation, child) {
      // Respect reduced motion
      if (MediaQuery.of(context).disableAnimations) {
        return child;
      }
      return FadeTransition(
        opacity: CurveTween(curve: AppAnimations.pageTransitionCurve)
            .animate(animation),
        child: child,
      );
    },
  );
}

CustomTransitionPage<void> _slidePage({
  required LocalKey key,
  required Widget child,
}) {
  return CustomTransitionPage<void>(
    key: key,
    child: child,
    transitionDuration: AppAnimations.slideDuration,
    reverseTransitionDuration: AppAnimations.slideDuration,
    transitionsBuilder: (context, animation, secondaryAnimation, child) {
      // Respect reduced motion
      if (MediaQuery.of(context).disableAnimations) {
        return child;
      }
      final tween = Tween(begin: const Offset(1.0, 0.0), end: Offset.zero)
          .chain(CurveTween(curve: AppAnimations.pageTransitionCurve));
      return SlideTransition(
        position: animation.drive(tween),
        child: child,
      );
    },
  );
}
```

```dart
  },

```
