## Warnings

### WR-01: Missing `mounted` check after `await` in `_resendCode()`

**File:** `mobile/lib/features/auth/screens/login_screen.dart:140`
**Issue:** After `await ref.read(authProvider.notifier).requestCode(...)`, `setState` is called on line 140 without checking `mounted`. If the user navigates away during the request, this throws `setState() called after dispose()`. The other async methods (`_requestCode`, `_verifyCode`) correctly check `mounted` before calling `setState` or showing UI, but `_resendCode` was missed.
**Fix:**
```dart
Future<void> _resendCode() async {
  if (!_canResend) return;
  setState(() => _isSubmitting = true);
  await ref
      .read(authProvider.notifier)
      .requestCode(_emailController.text.trim());
  if (!mounted) return;
  setState(() => _isSubmitting = false);
  _startResendCountdown();
}
```

### IN-01: Undisposed `FocusNode()` created inline in `KeyboardListener`

**File:** `mobile/lib/features/auth/screens/login_screen.dart:413`
**Issue:** Each OTP digit field wraps a `KeyboardListener` with `focusNode: FocusNode()` created inline. These FocusNodes are never disposed, causing a minor resource leak. In practice this is 6 FocusNodes that live for the widget's lifetime, so impact is minimal — but it's technically incorrect.
**Fix:** Either use a second list of FocusNodes (disposed in `dispose()`) or switch to the `Focus` widget which doesn't require managing a separate FocusNode:
```dart
// Option 1: Reuse existing code focus nodes or create a dedicated list
final _keyListenerFocusNodes = List.generate(6, (_) => FocusNode());

// In dispose():
for (final f in _keyListenerFocusNodes) { f.dispose(); }

// In build:
KeyboardListener(
  focusNode: _keyListenerFocusNodes[index],
  ...
)
```

### IN-02: Redundant consecutive spacers after logo

**File:** `mobile/lib/features/auth/screens/login_screen.dart:257-258`
**Issue:** Two `SizedBox` spacers appear back-to-back (`AppSpacing.md` then `AppSpacing.xl`) with nothing between them. This is likely a remnant from when a widget existed between them (possibly the old logo text or the removed JPEG). They could be combined into a single spacer for clarity.
**Fix:**
```dart
// Replace lines 257-258:
const SizedBox(height: AppSpacing.md),
const SizedBox(height: AppSpacing.xl),

// With a single spacer (combined height):
const SizedBox(height: AppSpacing.xl + AppSpacing.md),
// Or simply keep one:
const SizedBox(height: AppSpacing.xl),
```
